# 出海记录本

记录每次出海钓鱼的行程、地点、鱼获与海况。

## 功能特点

- **快速记录** — 日期、区域、出发/回港时间、人数、海况、结果、鱼种、备注
- **常用区域管理** — 自定义常用区域，一键点选填入
- **区域分析** — 自动统计每个区域的出钓次数、打龟率、最近记录
- **战报分享** — 一键复制格式化的出海战报，适合发微信群
- **草稿箱** — 意外退出不会丢失正在填写的内容
- **数据备份** — 导出/导入 JSON 格式备份，换手机也不怕丢数据
- **无障碍支持** — 提供标准/大/特大三种字体大小，方便不同视力需求

## 技术栈

- 纯 HTML + CSS + JavaScript，无任何框架依赖
- 数据存储在浏览器 localStorage
- 移动端优先的响应式设计
- 支持 PWA 可添加到手机桌面

## 关于

这是一个实用的小工具，用来帮助记录出海钓鱼的点点滴滴，方便日后回顾和总结渔获规律。

## 演示

直接在浏览器打开 `index.html` 即可使用。
